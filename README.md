# Vending-machine
Vending machine v2

## どんな自販機?
販売物は４つ。
1. triviaをランダムで
2. yesかnoか(ごく稀にmaybe)をランダムで
3. 各国のhelloをランダムで
4. 犬の写真をランダムで

## 使用API
- Dog API(https://dog.ceo/api/breeds/image/random)
- Hello, sault!(https://fourtonfish.com/hellosalut/?lang=ja)
- Yes Or No?(https://yesno.wtf/api)
- Trivia API(https://opentdb.com/api.php?amount=10)
